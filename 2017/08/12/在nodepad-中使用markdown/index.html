在nodepad++中使用markdown

发表于 2017-08-12 | 分类于 心得体会

Notepad++内置并不支持Markdown的语法。

Markdown语法高亮

首先下载Markdown规则文件，然后可以通过Notepad++中的 语言 -> 自定义语言格式 -> 导入 来实现；也可以通过手动操作实现，点击 开始菜单 -> 运行(或者直接按 Win+R) -> 输入 %APPDATA%\Notepad++，将userDefineLang.xml放置到此目录（如果已有同名文件存在，则将下载文件的内容增加进去，方法参考https://github.com/Edditoria/markdown_npp_zenburn ），重启软件。

Markdown实时预览

这个比较简单，看这篇文章《写了一个Notepad++的markdown插件》（原文链接：http://blog.gclxry.com/%E5%86%99%E4%BA%86%E4%B8%80%E4%B8%AAnotepad%E7%9A%84markdown%E6%8F%92%E4%BB%B6/ ，已经无法访问了），下载里面的NppMarkdown插件（也可以下载我备份在百度云的），复制dll到Notepad++安装目录的plugins目录下，重启软件。

Notepad++安装插件的两个方法：一个是通过软件自带的Plugin Manager来查找可用插件，勾选安装；还有个就是直接下载插件的dll文件，复制到Notepad++安装目录的plugins目录下，再重启软件即可。

百度云下载：http://pan.baidu.com/s/1c08pOjQ
